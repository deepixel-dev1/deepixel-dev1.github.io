Detailed Description

AR1ing 인스턴스 생성 클래스 (안드로이드용)

DPFactoryForAndroid는 해당 오브젝트 인터페이스를 인스턴스로 생성하는 클래스 이다. 객체를 생성하는 과정에서 라이센스와 관련된 예외가 발생할 수 있다.

<android>
std::shared_ptr<dp::ar1ingnative::IAR1ing> g_ptrARing =
dp::android::DPFactoryForAndroid::CreateInstance<dp::ar1ingnative::IAR1ing>(
env, activity);
