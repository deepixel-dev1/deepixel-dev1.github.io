dp.pixie Namespace

This namespace contains the classes that have information about the Pixie.
Classes

DPPixieUnity
This class provides the infomation including 3D position and events of Pixie. DPInputModule is reponsible for updating the contents of this class. Adding a DPInputModule to a scene updates the Pixie information every frame. On the android, you get information from the user's hand, while the editor gets information from the mouse and keyboard.
