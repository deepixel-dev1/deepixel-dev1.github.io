DPBaseMenuItem Methods

The DPBaseMenuItem type exposes the following members.

Methods
  NameDescription
GetViewportPoint
Returns viewport position that is transformed from world space into viewport space.
IsSelectable
Returns true if state of this item is selectable, false otherwise. For example, returns false while this item is being populated. Then returns true if it finishes.
OnChildSelected
Called then the child item of this item is selected.
OnEnter
Called when the user points to the location of this item to select it.
OnExit
Called when the user cancels to point to the location of this item to select it.
OnPotentialSelect
It tells you the item cloest to the location the user is pointing to.
OnSelected
Called when the user select this item. This method called after ShouldBeSelected() method returns true.
ShouldBeSelected
Returns true if this item is ready to be selected, false otherwise. If you want to delay OnSelect() event, returns false. OnSelect() method called if true is returned.
